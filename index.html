Welcome to Crystal Master!

How to play

There are four crystals which, when clicked, add a score to your total. Each crystal is assigned a random value at the start of each game.

You are given a random number at the start of the game which you must match in order to win.

Don't go too high! You will lose the game if your total score goes above the random number given to you at the start of the game.
